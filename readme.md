## 1-install geocoder:
```bash
pip install geocoder
```

## 2-install httpx:
```bash

pip install httpx
```

## 3-install PyQt6:
```bash
pip install PyQt6 PyQt6-tools
```

## 4-install Fluent_window_widgets:
```bash
pip install pyqt6-fluent-widgets
```

## Or You Can Install Modules: 
```bash 
pip install -r requirements.txt
```
